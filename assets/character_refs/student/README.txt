生徒キャラクター参照画像の配置ルール

このフォルダに、生徒キャラクターの透過PNGを10枚配置してください。

必須条件
- 背景透過PNG
- バストアップ
- 長辺1024px以上推奨
- 同一キャラクター、同一衣装、同一画風で統一
- 既存作品キャラクターは禁止
- 商用利用可能な素材のみ使用

必須ファイル名
- student_01_neutral.png
- student_02_happy.png
- student_03_smile.png
- student_04_surprised.png
- student_05_confused.png
- student_06_explaining.png
- student_07_serious.png
- student_08_thinking.png
- student_09_sad.png
- student_10_angry.png

推奨運用
- 生徒は 05_confused と 04_surprised の品質を特に揃える
- 疑問を持つ側なので、リアクションが分かりやすい表情差分を優先する
- 表情差分は顔中心で作り、髪型・衣装・構図は極力固定する
