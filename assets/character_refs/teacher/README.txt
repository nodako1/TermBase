先生キャラクター参照画像の配置ルール

このフォルダに、先生キャラクターの透過PNGを10枚配置してください。

必須条件
- 背景透過PNG
- バストアップ
- 長辺1024px以上推奨
- 同一キャラクター、同一衣装、同一画風で統一
- 既存作品キャラクターは禁止
- 商用利用可能な素材のみ使用

必須ファイル名
- teacher_01_neutral.png
- teacher_02_happy.png
- teacher_03_smile.png
- teacher_04_surprised.png
- teacher_05_confused.png
- teacher_06_explaining.png
- teacher_07_serious.png
- teacher_08_thinking.png
- teacher_09_sad.png
- teacher_10_angry.png

推奨運用
- 先生は 06_explaining と 07_serious の品質を特に揃える
- 解説役なので、口の開き方や視線方向に説明している印象を持たせる
- 表情差分は顔中心で作り、髪型・衣装・構図は極力固定する
